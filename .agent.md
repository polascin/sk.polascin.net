# SK Agent – audit, bezpečnosť a technická optimalizácia

## Popis projektu

Projekt **sk.polascin.net** je slovenský lekársky referenčný web (PHP). Obsahuje:
- Hlavnú stránku (`index.php`) — prehľad obsahu v slovenčine
- Referenčnú knihu (`refbook.sk.php`) — slovenský lekársky referenčný obsah
- Šablónu štýlov (`styles.css`) — dizajn stránky
- Bloky obsahu (`blocks/`) — interné obsahové bloky (blokované pre robotov)
- Dokumentáciu (`docs/`) — interná dokumentácia (blokovaná)
- Súbory (`files/`) — prílohy a súbory (blokované)
- Favikony (`favicon/`, `favicon.ico`)
- SEO infraštruktúru (`robots.txt`, `sitemap.xml`)
- Bezpečnostný súbor (`robots.txt`, `.htaccess`) — HTTP hlavičky, blokovanie scrapers

## 🎯 Cieľ agenta

Autonómny agent na komplexnú technickú správu a optimalizáciu projektu sk.polascin.net. Agent vykonáva hĺbkovú analýzu PHP/HTML/CSS kódu, spúšťa **Trunk** linting, overuje bezpečnostné nastavenia (`.htaccess`), kontroluje SEO integritu, responzívny dizajn a vykonáva **slovenskú jazykovú korektúru** — primárny jazyk webu je slovenčina so slovenskou lekárskou terminológiou.

## 📋 Primárne úlohy

### 1. Bezpečnostný audit

- **HTTP bezpečnostné hlavičky v `.htaccess`:**
  - Prítomnosť: `X-Frame-Options`, `X-Content-Type-Options`, `HSTS`, `Referrer-Policy`, `Permissions-Policy`, `Cross-Origin-Opener-Policy`, `X-Permitted-Cross-Domain-Policies`
  - `Options -Indexes` — blokovanie directory listingu
  - `ServerSignature Off`
  - Blokovanie `blocks/`, `docs/`, `files/`, `.git/`, `.env`, `*.key`

- **PHP bezpečnosť:**
  - Výstupný XSS — `htmlspecialchars()` na všetkých dynamických výstupoch
  - Bezpečnosť include/require — žiadne user-controlled cesty
  - Kontrola SQL dotazov (ak existujú) — PDO prepared statements

### 2. PHP/HTML audit

- Validita HTML5 štruktúry, `lang="sk"` atribút na `<html>`
- Správne zanorenie a uzavretie tagov
- `alt` atribúty na všetkých `<img>` elementoch
- `aria-label`, `aria-hidden`, `role` na interaktívnych elementoch
- `rel="noopener noreferrer"` na `target="_blank"` odkazoch
- PHP syntaktická validita a logické chyby
- Kontrola include/require ciest
- Nepoužívané premenné a mŕtvy kód

### 3. SEO optimalizácia

- **Meta tagy:** `description`, `keywords`, `author`, `robots`, `lang`
- **Canonical URL** — správnosť naprieč stránkami
- **hreflang:** `sk` a `x-default` pre slovenský obsah
- **Open Graph** — `og:title`, `og:description`, `og:url`, `og:image`, `og:locale` = `sk_SK`
- **robots.txt:** blokovanie `blocks/`, `docs/`, `files/`, `.git/`, scrapers, `Crawl-delay`
- **sitemap.xml** — aktuálnosť `lastmod`, správnosť URL
- Konzistentnosť interných odkazov

### 4. Slovenská jazyková korektúra ⭐

- **Primárna úloha** — celý web je v slovenčine
- Gramatická korektúra — slovenská gramatika, diakritika (á, é, í, ó, ú, ý, ä, č, š, ž, ľ, ĺ, ŕ, ô, dz, dž)
- Slovenská lekárska terminológia — konzistentnosť názvov (nefrológia, dialýza, transplantácia obličiek, CKD, eGFR)
- Štýlistická korektúra — odborný, zrozumiteľný slovenský štýl
- Konzistentnosť UI textov — tlačidlá, nadpisy, navigácia
- Zachovať odborný medicínsky charakter textu

### 5. Responzívny dizajn

- Overenie všetkých `@media` breakpointov v `styles.css`
- Kontrola pretekania obsahu na úzkych obrazovkách (320px–480px)
- Navigácia a obsah — škálovanie na mobile
- Veľkosť touch targets (min. 44×44 px)
- Kontrola pevných šírok, ktoré môžu spôsobiť horizontálny scroll

### 6. Integrita odkazov

- Overenie interných odkazov (`blocks/`, `files/`) — ciele existujú
- Konzistentnosť URL v `sitemap.xml` a na stránkach
- Externé odkazy — `rel="noopener noreferrer"`, `target="_blank"`

### 7. Trunk — linting a formátovanie

Trunk nakonfigurovaný v `.trunk/trunk.yaml`.

**Nakonfigurované nástroje:**

| Nástroj | Čo kontroluje |
|---|---|
| `prettier` | HTML, CSS, JS, JSON, Markdown |
| `markdownlint` | `.agent.md`, README súbory |
| `trufflehog` | Secrets a citlivé reťazce |
| `oxipng` | Kompresia PNG (favicon PNG súbory) |
| `git-diff-check` | Trailing whitespace, merge-conflict markery |

**Príkazy:**
```bash
trunk check --diff
trunk fmt --diff
trunk check --filter=trufflehog
```

**Ignorované:** `blocks/**`, `docs/**`, `files/**`

## 🔍 Skúmané súbory

### Prioritná kontrola
```
/index.php         → Hlavná stránka
/refbook.sk.php    → Referenčná kniha (hlavný obsah)
/styles.css        → Dizajn, responzívnosť
/.htaccess         → Bezpečnostné HTTP hlavičky
/robots.txt        → SEO blokovanie, scrapers
/sitemap.xml       → Aktuálnosť, štruktúra
/.gitignore        → Ochrana citlivých súborov
```

### Vylúčené zo skenovania
- `blocks/` — interný obsah (noindex, blokovaný)
- `docs/` — interná dokumentácia
- `files/` — prílohy
- `favicon/` — binárne súbory (ikony)
- `.git/` — git interné súbory

## ✅ Triviálne opravy

Agent **môže automaticky opraviť** bez explicitného súhlasu:
- Preklepy a gramatické chyby v SK textoch
- Chýbajúci `alt` text na `<img>` elementoch
- Chýbajúci `rel="noopener noreferrer"` na `target="_blank"` odkazoch
- Chýbajúce `aria-hidden="true"` na dekoratívnych elementoch
- Nekonzistentné odsadenie (whitespace)
- Aktualizácia `lastmod` v `sitemap.xml`

Agent **musí požiadať o súhlas** pred:
- Zmenami štruktúry navigácie alebo obsahu
- Úpravami `refbook.sk.php` (hlavný obsah)
- Zmenami PHP logiky
- Zmenami ovplyvňujúcimi viacero súborov naraz

## ⚙️ Pracovný postup agenta

1. **Inicializácia:** Agent načíta všetky cieľové súbory
2. **Trunk check:** `trunk check --diff` — linting, secrets, git-diff-check
3. **Trunk fmt:** `trunk fmt --diff` — automatické formátovanie
4. **PHP/HTML audit:** Validita, sémantika, bezpečnosť, include cesty
5. **Bezpečnostné skenovanie:** `.htaccess` hlavičky, PHP XSS vzory
6. **SEO analýza:** Meta tagy, canonical, hreflang, sitemap, robots.txt
7. **Jazyková korektúra:** Slovenské texty, diakritika, medicínska terminológia
8. **Responzívny dizajn:** Breakpointy, overflow, touch targets
9. **Integrita odkazov:** Interné URL, kotvy, externé weby
10. **Automatické opravy:** Triviálne opravy + Trunk fmt výstupy
11. **Generovanie správy:** Výsledky s prioritou (kritická → nízka)
